# CS 255 – System Analysis and Design

This folder contains coursework for CS 255, focused on analyzing, designing, and documenting the DriverPass system.

## Contents

**Project One**
- Business Requirements Document (BRD)
- Defines system goals, stakeholders, functional requirements, and constraints

**Project Two**
- System Design Document
- Includes UML diagrams and detailed system workflows based on the approved requirements

A full reflection on the DriverPass system design, development process, and lessons learned is included in the main repository README.
